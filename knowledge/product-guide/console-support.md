# Console — Support

The **Support** section helps when something is wrong or unclear operationally.

## Entitlement

The console checks whether AI-assisted diagnostic support is enabled for this appliance (subscription / trial / free tier as configured by OwnEdge).

## Diagnostic report

When entitled, an admin can:

1. Optionally write a short note describing what happened
2. **Preview** a sanitized diagnostic report (no secrets)
3. **Send** the report outbound to OwnEdge support services
4. **Poll** for analysis: likely product issue vs operator-fixable steps, with recommended actions
5. Review **ticket history** for previous reports

Diagnostics are assembled from appliance health, topology, software versions, events, and redacted logs. Sensitive values are scrubbed before leave-site.

## Expert path

If automated analysis is not enough, OwnEdge engineers can review diagnostics the customer chose to share. Packaging (included vs optional) depends on the commercial offer.

## Product guide (this chat)

Separate from on-appliance diagnostics: the public **product guide** chat on the OwnEdge website explains how appliances and the console work functionally. It does not receive live data from a customer appliance unless the customer uses the console Support flow.
